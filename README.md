# study-flyio-express

Express で簡単に作成したアプリケーションを [fly.io](https://fly.io/)にデプロイする方法を学ぶ

## COMMAND

### 1. ログインする

`flyctl auth login`

### 2. デプロイする

`flyctl deploy`

### 3. ページを開く（任意）

`flyctl open`
